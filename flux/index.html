Flux for Azure DevOps

A mobile client for Azure DevOps to manage your Work Items, Sprints, and Boards on the go.

Download on Play Store

About Flux

Flux is designed for developers and managers who need to stay connected to their Azure DevOps projects. It provides a clean, native mobile interface for managing work items, tracking sprints, and viewing boards, with full offline support so you can work anywhere.

Key Features

Work Item Management

Create, edit, and update work items seamlessly from your device.

Sprint Tracking

Monitor sprint progress and manage backlog items on the move.

Offline Support

Access and update your work items even when you don't have an internet connection.

Board Views

Visualize your project's workflow with mobile-optimized board views.

Security & Privacy

Privacy-First

Read our Privacy Policy. Flux connects directly to Microsoft Azure DevOps servers.

Tech Stack

  • Android SDK
  • Kotlin
  • Azure DevOps REST API
  • SQLite (for Offline Cache)
