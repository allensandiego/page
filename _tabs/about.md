### App Support, Home Lab & Agentic Automation

To maintain high availability and quality, my applications are supported by automated monitoring and custom DevOps loops:
- **Monitoring & Bug Tracking:** Integrated Bugsnag and Firebase Crashlytics to capture exceptions, automatically routing bug reports to GitHub Issues.
- **Home Lab Infrastructure:** Run a self-hosted Proxmox VE server locally hosting virtual machines (VMs).
- **Agentic Dev Loop:** Deployed and configured an OpenClaw agent locally to periodically scan open GitHub Issues, assess the bugs, and report findings to a dedicated Slack channel. 
- **Human-in-the-Loop Workflow:** After reviewing the agent's Slack reports, I prompt the agent to implement the fix, run the project's automated test suite, and submit a Pull Request.
- **Delivery Pipeline:** Approving and merging the PR triggers the CI/CD pipeline, deploying the updated build to internal testing environments. Once manually verified, the release is promoted to production.
